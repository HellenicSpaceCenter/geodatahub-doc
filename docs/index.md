# GeoDataHub Documentation

> Υποσύστημα Διαχείρισης αιτημάτων & Διανομής Γεωχωρικών Δεδομένων

---

This guide provides comprehensive documentation for the Hellenic Space Center's GeoDataHub platform, covering both user and administrator perspectives. It includes detailed instructions, best practices, and reference materials to help you effectively use and manage the platform.

## Access 

The platform is available at [{{env.ERM_PRODUCTION_URL}}]({{env.ERM_PRODUCTION_URL}}). Access is restricted and requires authentication. Public usage is not permitted.

## Documentation Structure

The documentation is organized into the following main sections:

- [User Guide](./user-guide/index.md): Instructions and best practices for end-users of the GeoDataHub platform, including how to submit orders, track requests, and access data.
- [Admin Guide](./admin-guide/index.md): Detailed documentation for administrators managing the GeoDataHub platform, covering configuration, workflow management, credential storage, and more.


## Project status

The platform is currently under internal testing by the Hellenic Space Center team. As such, the documentation is a living resource that will be continuously updated and expanded as new features are developed and feedback is received. Development is also ongoing to integrate with the AXIS-3 (Governmental Hub) of the Greek National Satellite Space program.
