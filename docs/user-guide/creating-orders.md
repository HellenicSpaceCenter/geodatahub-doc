# Creating Orders

Step-by-step guide to creating satellite imagery orders in GeoDataHub ERM.

## Order Creation Workflow

1. **Start New Order** - Click the "New Order" button
2. **Select Product** - Choose satellite imagery product
3. **Define Area of Interest** - Draw or upload your geographic boundary
4. **Specify Parameters** - Set temporal range and processing options
5. **Review & Submit** - Verify details and submit for processing

## Product Selection

Browse available products:

- Sentinel-2 imagery
- Planet Labs imagery
- Custom processing services
- Stock imagery library

## Area of Interest

Define your study area:

- Draw on map
- Upload GeoJSON/Shapefile
- Enter coordinates
- Select predefined areas

## Order Parameters

Configure order-specific settings:

- Temporal range
- Data resolution
- Processing bands
- Output format
- Priority level

## Submission & Confirmation

Final steps before order submission:

- Review all parameters
- Set approval workflow
- Configure deliverables
- Submit order
