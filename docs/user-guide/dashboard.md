# Dashboard

The dashboard is your central hub for interacting with GeoDataHub ERM.

## Dashboard Overview

The main dashboard displays:

- Recent orders and their status
- Pending approvals and actions
- Quick access to key features
- System announcements and notifications

- Main menu

- Breadcrumb navigation
- Search functionality

- Order status summary
- Quick order creation
- Recent activities
