# Chat & Collaboration

Real-time communication and collaboration within GeoDataHub ERM.

## Chat Features

Built-in communication system:

- Order-specific chat rooms
- Team discussions
- Real-time messaging
- Message history
- File sharing in chat
- @mentions and notifications

## Starting a Conversation

Initiate collaboration:

- Click chat icon on order
- Create group chat
- Invite team members
- Find existing conversations

## Message Types

Different ways to communicate:

- Text messages
- Attachments and files
- Mentions (@username)
- Threading for organized discussions
- System messages and notifications

## Notifications

Stay informed:

- New message notifications
- @mention alerts
- Order updates
- Workflow stage changes
- File shared notifications

## Best Practices

Effective collaboration:

- Ask clarifying questions early
- Share relevant documents
- Use clear subject lines
- Mention responsible parties
- Keep discussions organized
