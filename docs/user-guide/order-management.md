# Order Management

Guide to managing and tracking your satellite imagery orders.

## Order Status Tracking

Monitor order progress through different stages:

- Draft
- Submitted
- Under Review
- Approved
- Processing
- Ready for Delivery
- Delivered
- Completed

## Order Details

View and manage individual order information:

- Order metadata
- Specifications
- Timeline
- Cost estimate
- Associated team members

## Order Actions

Available actions at each stage:

- Edit draft orders
- Cancel orders
- Request more information
- Approve orders (if supervisor)
- Download deliverables
- Reorder with same parameters

## Search & Filter

Find specific orders:

- Search by order ID
- Filter by status
- Filter by date range
- Filter by project
- Filter by team

## Order History

Track order evolution:

- Status changes
- Comments and notes
- Time tracking
- Cost tracking
