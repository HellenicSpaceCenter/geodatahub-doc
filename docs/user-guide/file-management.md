# File Management

Uploading, organizing, and managing files in GeoDataHub ERM.

## File Types

Supported file types:

- Geospatial data (GeoTIFF, GeoJSON, Shapefile, etc.)
- Documents (PDF, Word, Excel, etc.)
- Images (JPEG, PNG, etc.)
- Archives (ZIP, TAR, etc.)
- Custom formats

## Uploading Files

Add files to orders:

1. Click upload button
2. Select file(s) from computer
3. Set file metadata
4. Confirm upload
5. Monitor progress

## File Organization

Organize your files:

- Create folders/directories
- Group by project
- Tag files
- Search files
- Version control

## File Sharing

Share files with team:

- Share to specific users
- Share to groups
- Set permissions (view, edit, download)
- Manage access
- Track file access

## Downloading Files

Retrieve files:

- Download single files
- Batch download
- Export as archive
- Share download links
- Set download expiration
