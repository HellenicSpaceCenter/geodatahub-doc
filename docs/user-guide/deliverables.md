# Deliverables

Managing and accessing order deliverables and results.

## What Are Deliverables?

Processed data and results from orders:

- Satellite imagery
- Processed datasets
- Analysis reports
- Metadata files
- Derived products

## Deliverable Status

Track deliverable preparation:

- Processing in progress
- Ready for review
- Ready for delivery
- Delivered
- Archived

## Accessing Deliverables

Retrieve your results:

1. Navigate to order
2. Go to Deliverables section
3. View available deliverables
4. Download or preview
5. Access metadata

## Deliverable Details

Information for each deliverable:

- File name and size
- Creation date
- Data format
- Quality metrics
- Processing parameters
- Checksums for verification

## Download & Export

Retrieve deliverables:

- Download individual files
- Download all as archive
- Export metadata
- Generate reports
- Share link

## Deliverable Lifecycle

From creation to archival:

- Generation
- Quality assurance
- Publication
- Distribution
- Retention and archiving
